🚆 Mumbai Local Railway Passenger Traffic & Congestion Analysis Dashboard

Overview
This Power BI dashboard analyzes passenger traffic patterns across Mumbai Local Railway stations.

Key Features
- Executive Overview Dashboard
- Peak Hour Analysis
- Station Utilization Analysis
- Interactive Filters
- KPI Tracking
- Heatmap Analysis
- DAX Measures

Tools Used
- Power BI
- DAX
- Power Query
- Excel

Dashboard Pages
Executive Overview
- Total Passengers
- Daily Average Passengers
- Busiest Station
- Peak Hour
- Busiest Route

Peak Hour Analysis
- Peak Passenger Count
- Morning Traffic Share
- Hourly Traffic Trends
- Heatmap Analysis

Station Utilization Analysis
- Most Crowded Station
- Least Utilized Station
- Occupancy Analysis
- Station Ranking

Dataset
3000+ simulated Mumbai Local Railway passenger records including:
- Date
- Railway Line
- Passenger Count
- Occupancy Rate
- Delay Minutes
- Train Type
- Weather Conditions
